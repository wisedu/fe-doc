# LayoutSpacing

> 空白间距行。

------------

## 引入

```javascript
import { LayoutSpacing } from 'mint-ui';

Vue.component(LayoutSpacing.name, LayoutSpacing);
```

## 例子
背景

```html
<mt-layout-spacing background="grey"></mt-layout-spacing>
```

高度

```html
<mt-layout-spacing height="20px"></mt-layout-spacing>
```


## API

| 参数 | 说明 | 类型 | 可选值 | 默认值 |
|------|-------|---------|-------|--------|
| background | 背景 | String |  |  |
| height | 高度 | String |  | |
